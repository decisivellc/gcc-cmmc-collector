How do I find these values?

You need a GCC-High Entra app registration with application-level Graph permissions. First-time setup is ~20 minutes and requires a Global Administrator (or Application Administrator + Privileged Role Administrator) in the tenant.

  1. Go to entra.microsoft.us → Applications → App registrations → + New registration. Name it cmmc-gcc-evidence-collector, single-tenant, no redirect URI.
  2. On the app's Overview page, copy Application (client) ID and Directory (tenant) ID.
  3. Under Certificates & secrets → + New client secret. Copy the Value once — you will not see it again.
  4. Under API permissions → Add a permission → Microsoft Graph → Application permissions, add: User.Read.All, Directory.Read.All, AuditLog.Read.All, Policy.Read.All, IdentityRiskEvent.Read.All, DeviceManagementManagedDevices.Read.All, DeviceManagementConfiguration.Read.All, SecurityEvents.Read.All, ThreatIndicators.Read.All, Sites.Read.All.
  5. Click Grant admin consent for <tenant>. Every row should flip to Granted. Missing consent is the #1 cause of 403 errors.

Full walkthrough: SETUP.md. API-driven provisioning is tracked as a separate ticket.
